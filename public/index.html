ELVA Notify Service

Internal communication service for OTP and notifications via SMS and Email.

Base URL

https://notify.elvatech.in

Authentication

Use API Key in header:

x-api-key: <your-api-key>

OTP APIs

  • POST /otp/send
  • POST /otp/resend
  • POST /otp/verify
{ "phone": "9876543210" }

Notification API

  • POST /notify/send
{ "phone": "9876543210", "message": "Your task is approved" }

Request Rules

  • Provide either phone OR email (not both)
  • API key required in header
  • OTP expires in 5 minutes

Sample CURL

        curl -X POST https://notify.elvatech.in/otp/send \
        -H "Content-Type: application/json" \
        -H "x-api-key: YOUR_API_KEY" \
        -d '{"phone":"9876543210"}'
